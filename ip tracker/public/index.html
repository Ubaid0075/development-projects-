Visitor Access Monitor

Modern landing page with access logging

This page is designed to greet visitors with a clean, simple experience while sending a server-side notification when the page loads.

What happens

When a visitor opens this page, the browser calls a Vercel serverless API route that records the visitor IP address, timestamp, and user-agent, then sends an email notification.

Privacy notice

By continuing to use this page, you acknowledge that access information may be recorded for operational and security purposes.
